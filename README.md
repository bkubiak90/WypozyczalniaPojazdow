# Projekt wypożyczalni samochodowej

## Autorzy
Jan Wądołowski i Bartłomiej Kubiak
sem. II, R.A. 2019/20

## Opis
Program umożliwia dodawanie do osobnych baz danych pojazdy, wypożyczenia oraz klientów (wraz z ich adresami), umożliwia wypożyczenie oraz zwrot pojazdu, wyliczenie kosztu danego wypożyczenia, jak również przydzielenia klientowi rabatu zgodnie z określonymi kryteriami (typ klienta).
